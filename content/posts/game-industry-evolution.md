아타리 쇼크 이후 게임 산업은 살아남았다. 그것도 훨씬 더 크게. 2023년 전 세계 게임 시장은 약 1,840억 달러다. 그런데 플레이어들은 점점 더 많은 것을 빼앗기는 느낌을 받는다. 게임 하나를 사는 것으로 끝나던 시절이 있었는데, 이제는 사고 나서도 계속 돈이 나간다. 이것은 단순히 기업이 탐욕스러워진 것이 아니다. 구조가 바뀐 것이다.

---

## 1시대: 아케이드 — 플레이당 과금 (1970s~1980s)

게임 산업의 원래 수익 모델은 명확했다.

```
동전 투입 → 플레이 → 게임 오버 → 동전 다시 투입
```

업소용 기계를 만들어 오락실에 팔거나 임대하고, 오락실은 손님에게 1회당 돈을 받는다. 게임이 재미있을수록 동전이 많이 들어온다. **재미 = 수익**의 등식이 가장 직접적으로 연결된 시대였다.

이 모델의 특성:
- 개발사는 기계를 팔면 끝 (하드웨어 판매)
- 오락실 운영자가 플레이어에게 과금
- 어렵게 만들수록 동전이 많이 들어오는 구조 → 난이도 인플레이션

아케이드 게임이 일부러 어려운 이유가 여기 있다. 미숙한 플레이어가 빨리 죽을수록 더 많은 동전을 넣는다.

## 2시대: 패키지 판매 — 완전한 제품을 한 번에 (1977~1990s)

아타리 2600이 만든 가정용 게임기 시장, 패미컴이 재건한 시장, PC 게임 시장 모두 같은 모델이었다.

```
개발 → 패키지 완성 → 유통 → 소비자가 구매 → 거래 종료
```

소비자가 지불한 돈으로 게임 전체를 소유했다. 게임이 출시된 이후에는 개발사와 소비자 사이에 더 이상의 금전 거래가 없었다. 패치도 없었다. 버그가 있으면 그냥 있는 것이다. 다음 버전을 사면 됐다.

**이 시대의 수익 공식:**

```
판매량 × 개당 마진 = 수익
```

더 많이 팔려면 더 좋은 게임을 만들어야 했다. 마케팅과 입소문이 핵심이었다. 소비자와의 관계는 구매 순간에 끝났다.

**게임 잡지의 탄생:**

소비자가 구매 전에 게임 내용을 알 수 없었기 때문에 전문 리뷰 잡지가 권력을 가졌다. 패미통, EGM, GameFan 같은 잡지의 점수가 판매량을 결정했다. 리뷰어를 관리하는 PR이 중요해진 것도 이 시대다.

## 3시대: 확장팩과 CD-ROM — 제품의 분리 시작 (1990s)

CD-ROM의 등장으로 용량이 폭발했다. 게임의 규모가 커졌고, 이야기가 길어졌고, 제작비가 올라갔다. 동시에 **확장팩(Expansion Pack)**이라는 개념이 자리잡았다.

```
본편 구매 (39,900원) + 확장팩 구매 (29,900원) = 완전한 경험
```

스타크래프트의 브루드워, 디아블로 2의 파괴의 군주, 심즈의 수십 개 확장팩. 이것은 완전히 새로운 게임을 추가하는 형태였다. 본편을 산 사람이 추가 콘텐츠를 원하면 확장팩을 사면 된다는 개념은 납득 가능했다.

**PC 온라인 RPG의 탄생 — 구독 모델 등장:**

1996년 울티마 온라인, 1999년 에버퀘스트가 월정액 구독 모델을 확립했다.

```
월 $9.99~$14.99 구독 → 지속적으로 콘텐츠가 업데이트되는 서비스
```

2004년 월드 오브 워크래프트(WoW)가 이 모델의 정점이었다. 최고 구독자 수 1,200만 명, 월 $15. 이 모델은 소비자에게도 명확했다: **돈을 내면 서비스를 받는다.** 서비스가 끊기면 돈도 끊는다. 상대적으로 투명했다.

## 4시대: 디지털 배포와 DLC — 분리의 가속 (2000s)

Xbox Live(2002)와 PlayStation Network, Steam(2003)이 열리면서 패키지 없이 게임을 팔 수 있게 됐다. 그리고 **DLC(Downloadable Content)**가 등장했다.

### 말의 갑옷 사건 (2006)

```
엘더스크롤 4: 오블리비언
추가 콘텐츠: 말 갑옷 (Horse Armor Pack)
가격: $2.50
내용: 말 외형 변경
게임 플레이에 대한 영향: 없음
```

이것이 최초의 주요 유료 코스메틱 DLC 논란이다. 개발사 베데스다는 욕을 먹었다. 그러나 팔렸다. 이후 DLC 모델이 산업 전체로 확산됐다.

**DLC의 진화:**

```
2006년 초기: 코스메틱 위주 (말 갑옷)
2007~2010년: 스토리 DLC (기존 게임 내용 확장)
2010년~: 온라인 패스 (중고 판매 차단용 코드 동봉)
      → 온라인 기능을 쓰려면 새 코드 필요
      → 중고로 사면 코드 없음 → 추가 구매
```

**온라인 패스**는 게임 구조를 직접 건드린 첫 번째 사례였다. 중고 시장에서 개발사가 돈을 못 받는 구조를 억지로 바꿨다. 결국 EA가 2013년 폐지했지만, 소비자의 반발로 철회됐을 뿐 시도는 계속됐다.

## 5시대: 모바일과 F2P — 수익 구조의 근본 변환 (2007~)

2007년 아이폰이 등장했다. 앱스토어가 열렸다. 게임 가격의 기준이 붕괴됐다.

```
PC/콘솔 게임: $30~$60
앱스토어 게임: $0.99, $1.99
→ 곧: 무료 (Free to Play, F2P)
```

**왜 F2P가 됐나:**

앱스토어에서 소비자는 가격이 조금이라도 있으면 다운로드를 꺼렸다. 무료가 기본이 됐다. 그러면 어떻게 수익을 내는가.

```
F2P 수익 공식:
  매우 많은 무료 사용자 중
  소수의 "고래(Whale)"가 대부분의 매출을 만든다

전형적인 분포:
  전체 사용자의 50%: $0 지출
  전체 사용자의 47%: $1~$20 지출
  전체 사용자의 2.9%: $20~$100 지출
  전체 사용자의 0.1%: $1,000+ 지출
  → 이 0.1%가 전체 매출의 30~50%를 만들기도 함
```

이 구조에서 게임 설계의 목표가 바뀐다.

```
패키지 시대 목표: 재미있어서 많은 사람이 사게 만들기
F2P 시대 목표:   고래가 많이 쓰게 만들기
                 + 무료 사용자도 잃지 않기 (그래야 고래의 경쟁 대상이 있음)
```

### 가챠의 구조

가챠(Gacha)는 일본의 가챠폰(캡슐 뽑기 기계)에서 온 용어다. 게임에서는 확률 기반 아이템 획득 시스템이다.

```
10,000원 → 랜덤 아이템 획득
원하는 아이템이 나올 확률: 1~5%
원하는 아이템을 얻으려면 평균: 20~100회 시도
평균 비용: 20만원~100만원
```

이것이 도박과 같은 심리 메커니즘을 사용하는 이유다.

```
도박의 강화 원리:
  가변 비율 강화 (Variable Ratio Reinforcement)
  → 언제 보상이 올지 모를 때 행동이 가장 강화됨
  → 슬롯머신과 동일한 원리
  → 가챠도 같음
```

가챠는 일본에서 시작해 한국(2000년대 온라인 게임 아이템 강화), 전 세계 모바일로 퍼졌다. 원신(Genshin Impact)이 2020년 이것을 AAA 수준 그래픽과 결합해 전 세계 표준으로 만들었다.

## 6시대: GaaS — 게임이 서비스가 되다 (2010s~현재)

GaaS(Games as a Service)는 단순히 DLC가 많은 것이 아니다. 게임 자체가 지속적으로 운영되는 플랫폼이 되는 것이다.

### 배틀패스 — 산업의 표준이 된 모델

2017년 포트나이트가 배틀패스(Battle Pass)를 도입하며 업계 표준이 됐다.

```
배틀패스 구조:
  시즌 1 (3개월): $10 구매
  → 미션을 완료하며 레벨업
  → 레벨업할수록 보상(코스메틱) 지급
  → 시즌 종료 → 보상 사라짐 → 시즌 2 배틀패스 구매

소비자 심리:
  "$10만 내면 100가지 보상을 받는다"
  → 사실상 시즌마다 $10 구독

기업 입장:
  예측 가능한 정기 매출
  + 보상을 얻으려는 플레이 지속 유인
  + 프리미엄 배틀패스, 번들 등 추가 판매 기회
```

배틀패스가 영리한 이유는 **FOMO(Fear of Missing Out)**를 시스템화했기 때문이다.

```
FOMO 설계:
  시즌 한정 아이템 → "이번 시즌에만"
  → 안 사면 영원히 못 얻음
  → 당장 사지 않으면 손해라는 심리
```

배틀패스를 채택한 게임: 포트나이트, 에이펙스 레전드, 콜 오브 듀티, 발로란트, 리그 오브 레전드(패스 형태로), PUBG, Dota 2... 사실상 모든 멀티플레이어 라이브 서비스 게임.

### 시즌 콘텐츠와 유저 잡아두기

GaaS 게임은 플레이어가 게임을 떠나지 않도록 설계된다.

```
잡아두는 장치들:
  데일리 퀘스트: 매일 접속해야 보상
  위클리 미션: 주마다 접속해야 달성
  시즌 배틀패스 레벨: 안 하면 구매한 돈이 아깝다
  길드/클랜 활동: 내가 빠지면 팀원에게 피해
  한정 이벤트: 지금 해야 함
```

이것들은 "재미있어서" 접속하는 것이 아니라 "해야 해서" 접속하는 구조다.

## 근본적인 구조 변화

여기서 과거와 현재의 구조적 차이가 명확해진다.

### 과거의 구조

```
[개발사] → 완성된 제품 → [유통] → [소비자]
           ↑                           ↓
      일회성 거래                   소유권 이전
      
수익 동기: 좋은 게임을 만들어 많이 팔기
```

### 현재의 구조

```
[게임사] → 플랫폼(게임) → [플레이어]
              ↑                  ↓
         지속적 업데이트    지속적 참여/지출
         
수익 동기: 플레이어를 플랫폼 안에 묶어두고
          지속적으로 지출하게 만들기
```

이 두 구조의 차이가 현재 플레이어들이 "빼앗긴다"는 느낌을 받는 근원이다.

**과거**: 게임은 완성된 제품이었다. 소비자는 가치를 정확히 알고 돈을 냈다.

**현재**: 게임은 진입점이다. 실제 수익은 이후 지속적인 소비에서 나온다. 진입 비용(게임 가격)을 낮추거나 없애는 대신, 내부에서 더 많은 돈을 쓰게 설계된다.

## 기업이 유저에게서 "빼앗는" 방법들

### 콘텐츠 분리 — 예전이라면 기본이었을 것들

```
2000년대 게임:
  캐릭터 스킨 → 게임 내에서 달성하면 해금
  스토리 추가 → 확장팩 (새로운 게임 분량)
  신규 맵 → 패치로 추가

현재:
  캐릭터 스킨 → 가챠 또는 현금 구매
  스토리 → 시즌 패스 포함 또는 별도 DLC
  신규 맵 → 배틀패스 구매 필요하거나 별도 판매
```

예전에 게임 내 달성 보상으로 있었을 콘텐츠가 유료 상품이 됐다. 이것이 "빼앗는다"는 느낌의 근원이다. 빼앗은 게 아니라 원래부터 없는 것이지만, 플레이어 세대가 이미 "당연히 있어야 한다"고 기억하기 때문이다.

### 프리미엄 통화 — 심리적 거리두기

```
실제 돈 → 다이아/크리스탈/젬/코인(가상 화폐) → 아이템

직접 비교:
  아이템 = $9.99  → "비싸다"
  아이템 = 1,000 다이아 → "잘 모르겠다"
```

현금을 게임 내 통화로 바꾸면 소비자가 실제 금액을 즉각적으로 인식하기 어려워진다. 통화 단위가 달라 환산을 해야 하고, 그 계산이 귀찮아 그냥 쓰게 된다.

또한 "900 다이아" 패키지를 파는데 아이템이 "1,000 다이아"면 100이 모자란다. 자연스럽게 "2,000 다이아" 패키지를 사게 된다. 남은 1,000 다이아는 또 다음 소비를 유도한다.

### 강화와 확률 — 매몰 비용 이용

```
한국 온라인 게임이 발전시킨 강화 시스템:
  +1 강화: 100% 성공
  +5 강화: 80% 성공
  +8 강화: 40% 성공
  +10 강화: 15% 성공
  → 실패하면 강화 수치가 내려가거나 아이템이 파괴됨
```

실패할 때마다 "여기까지 왔는데"라는 매몰 비용 심리가 작동한다. 이미 쓴 돈이 아까워서 더 쓴다. 이것은 도박의 근본 원리다.

### 사회적 압박 — 랭킹과 비교

멀티플레이어 게임에서 더 강한 캐릭터, 더 희귀한 스킨은 다른 플레이어에게 보인다.

```
PvP 게임에서 돈 쓴 플레이어 > 안 쓴 플레이어 (P2W)
  → 이기고 싶으면 돈을 써야 함

코스메틱 게임에서 희귀 스킨을 가진 플레이어
  → 부러움, 열등감, 따라 사고 싶은 욕구
```

소셜 비교가 소비를 유도한다. 이것은 명품 산업의 논리와 동일하다.

### 시간 기반 제한 — FOMO의 무기화

```
한정 판매:
  "이번 주말만 판매하는 스킨"
  "이번 시즌에만 획득 가능한 아이템"
  "오늘 자정까지만"

자연스러운 반응:
  "나중에 사자" → 안 됨
  "생각해보자" → 시간이 없음
  → 충동 구매 유도
```

이것은 쇼핑몰의 "오늘만 이 가격" 마케팅과 동일하다. 다만 게임에서는 정말로 그 아이템이 영원히 사라진다(아니면 몇 달 후 다시 나온다).

## 왜 이렇게 됐나 — 제작비와 투자자

이 모든 변화에는 구조적 원인이 있다.

### AAA 게임 제작비의 폭발

```
1990년대 게임 제작비: 수십만 달러~수백만 달러
2000년대: 수천만 달러
2010년대: 수억 달러
2020년대: 
  사이버펑크 2077: 약 3억 1,600만 달러
  레드 데드 리뎀션 2: 약 5억 달러 추정
  Grand Theft Auto 6: 수억 달러 이상
```

제작비가 5억 달러면 $70짜리 게임으로 본전을 뽑으려면 700만 장 이상을 팔아야 한다. 유통, 마케팅, 개발자 인건비까지 더하면 실제 손익분기점은 훨씬 높다. 한 번의 판매로는 불가능한 숫자다.

### 상장 기업과 주주 압박

```
패키지 게임의 매출 패턴:
  출시 첫 달: 폭발적 매출
  이후: 빠르게 감소
  → 분기 실적이 고르지 않음
  → 주주 입장에서 예측 불가능

구독/GaaS 매출 패턴:
  매달 일정한 수익
  → 예측 가능한 MRR(Monthly Recurring Revenue)
  → 주주가 좋아함, 밸류에이션 높아짐
```

상장된 게임사에게 주주의 압박은 강력하다. "지속적 수익 구조"를 만들어야 한다는 요구가 GaaS로의 전환을 가속했다.

### 데이터의 힘

인터넷 연결 게임은 모든 것을 데이터로 기록한다.

```
어느 구간에서 플레이어가 이탈하는가
어떤 아이템에 얼마를 쓰는가
어떤 FOMO 메시지가 구매 전환율을 높이는가
어떤 난이도에서 과금 전환이 많이 일어나는가
```

이 데이터를 기반으로 "최적화"된 과금 설계가 가능하다. 플레이어의 지갑을 가장 효과적으로 열 수 있는 지점을 과학적으로 찾아낸다. 이것이 현대 게임의 "게임 디자인"의 일부다.

## 현재의 흐름 — 2020년대

### 구독 서비스의 부상

```
Xbox Game Pass: 월 $10~$15, 수백 개 게임 플레이 가능
PlayStation Plus: 월 $10~$18, 구독 게임 포함
Apple Arcade: 월 $7, 광고/과금 없는 게임
Netflix 게임: 구독자 포함, 별도 과금 없음
```

이 모델은 반대 방향처럼 보인다. 한 번 내고 여러 게임을 즐긴다. 하지만 기업 입장에서는 예측 가능한 구독 수익이다. 게임 하나의 판매가 아닌, 플레이어를 플랫폼에 가두는 전략이다.

### 규제의 등장

```
2018년 벨기에: 루트박스를 도박으로 분류, 판매 금지
2019년 네덜란드: 루트박스 규제 (일부 게임 서비스 종료)
중국: 미성년자 게임 시간 및 과금 제한
한국: 확률형 아이템 확률 공개 의무화
```

처음으로 외부에서 게임 과금 구조에 개입이 시작됐다. 업계가 자율 규제를 못 하면 정부가 개입한다.

### 팬들의 반격

```
Diablo Immortal (2022):
  블리자드의 모바일 게임
  최대 110,000달러를 써야 최고 캐릭터 완성 가능
  출시 즉시 리뷰 폭탄, 밈이 됨

BF2042, Halo Infinite, 다수 EA 게임:
  과도한 과금에 대한 집단적 반발
  → 일부 정책 철회
```

플레이어들이 집단으로 반발하고, 미디어가 이를 증폭시키고, 주가에도 영향을 주는 일이 반복됐다. 반발이 있다는 것은 아직 시장이 완전히 굴복하지 않았다는 의미이기도 하다.

## 정리 — 구조 변화의 지도

```
시대        수익 모델         소비자와의 관계
────────────────────────────────────────────
아케이드     회당 과금          즉각적, 투명
패키지       일회성 판매        구매=소유, 종료
확장팩       추가 구매          선택적, 납득 가능
DLC          분리 판매          시작은 납득, 점점 남용
구독 MMO     월정액             투명, 서비스 대가
모바일 F2P   가챠/고래          게임이 도박화
GaaS/배틀패스 반복 구독+FOMO   심리 설계, 지속 지출
2020s        구독+GaaS 혼재    플랫폼 종속

─────────────────────────────────
핵심 변화:
  완성된 제품을 파는 것 → 플레이어를 플랫폼에 가두고 지속적으로 과금
  재미로 오게 만들기 → 떠나기 어렵게 만들기
  좋은 게임 = 수익 → 지속적 지출 = 수익
```

기업이 유저에게서 "빼앗는다"는 느낌의 근원은 이 구조 변화다. 플레이어는 완성된 제품을 기대하는데, 기업은 플레이어를 지속 수익의 원천으로 본다. 이 기대의 불일치가 매 논란의 뿌리다.

이것은 탐욕의 문제이기도 하지만, 더 근본적으로는 제작비, 투자자 압박, 데이터 기반 최적화가 만들어낸 구조의 문제다. 구조가 바뀌지 않으면 개별 기업이 "착한 척"해도 시장 압력이 같은 방향으로 밀어붙인다.
