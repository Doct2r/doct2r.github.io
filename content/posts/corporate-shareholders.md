"기업은 이윤을 추구한다." 경영학 교과서의 첫 줄이다. 틀린 말은 아니다. 하지만 절반만 맞다. 정확하게는 이렇다. **기업은 특정 주주가 원하는 방향으로 움직이며, 그 결과로 이윤이 발생하거나 발생하지 않는다.** 누가 최대 주주냐에 따라 회사의 방향, 문화, 생존 여부가 결정된다. 그리고 방향이 틀려서 회사가 망해도, 결정을 내린 사람이 퇴장하는 경우는 생각보다 드물다.

---

## 1. 기업은 무엇이고 무엇을 추구하는가

### 교과서의 답

밀턴 프리드먼(Milton Friedman)은 1970년 뉴욕타임스 기고에서 이렇게 썼다.

> "기업의 사회적 책임은 하나다. 규칙 안에서 이윤을 극대화하는 것."

이것이 **주주 자본주의(Shareholder Capitalism)**다. 20세기 후반 미국 경영의 지배적 이념이 됐고, 지금도 기본값으로 작동한다.

이 논리에 따르면:

```
기업 목표: 주주 가치(Shareholder Value) 극대화
  → 주가 상승 + 배당 증가
  → CEO 평가 기준 = 주가 + EPS(주당순이익)
  → 모든 결정은 이 기준으로 수렴
```

### 그런데 실제로는

이윤 극대화라는 목표는 추상적이다. 누가 어떤 방식으로 이윤을 추구할지를 결정하는 것은 **인간**이다. 그리고 그 인간은 주주, 그중에서도 지배력을 가진 **최대 주주**다.

같은 "이윤 추구" 명령을 받아도:

```
최대 주주가 단기 수익을 원하면:
  → 비용 절감, 감원, 자사주 매입, 배당 확대
  → R&D와 인력 투자 축소

최대 주주가 장기 성장을 원하면:
  → 제품 투자, 인재 확보, 시장 점유율 확장
  → 단기 이익 희생 감수

최대 주주가 생태계 지배를 원하면:
  → 적자 감수 + 경쟁사 압살 + 나중에 독점 수익
  → 아마존, 우버의 초기 전략
```

"이윤 추구"는 방향이 아니다. 방향을 결정하는 것은 지배주주의 의지다.

## 2. 기업을 조종하는 자는 누구인가

### 주주의 종류

모든 주주가 동등한 영향력을 갖지 않는다.

```
기관 투자자 (Institutional Investors):
  뱅가드(Vanguard), 블랙록(BlackRock), 피델리티
  → 수십 개 회사에 분산 투자
  → 개별 회사 경영보다 시장 전체 안정에 관심
  → "조용한 주주" — 개입보다 보유

행동주의 투자자 (Activist Investors):
  칼 아이칸(Carl Icahn), 엘리엇 매니지먼트(Elliott Management)
  → 지분 집중 매입 후 경영 압박
  → 요구 사항 명확: 비용 절감, 분사(Spin-off), 자사주 매입
  → "시끄러운 주주" — 개입이 목적

지배주주 / 창업자:
  버크셔 해서웨이의 워런 버핏
  알파벳의 래리 페이지 (클래스 B 주식으로 의결권 집중)
  소프트뱅크의 손정의
  → 회사의 방향을 직접 결정
  → 주가보다 비전이 우선되기도 함
```

### 의결권 구조가 핵심이다

주식을 많이 가지는 것만으로 지배력이 생기지 않는다. **의결권(Voting Rights)**이 중요하다.

```
일반 주식: 1주 = 1표

차등 의결권 구조:
  구글(알파벳): 클래스 A(1표) / 클래스 B(10표) / 클래스 C(0표)
  메타: 마크 저커버그 클래스 B 보유 → 사실상 지배
  스냅(Snap): 일부 주식은 의결권 0

결과:
  구글 공동창업자들은 지분율이 낮아도
  → 의결권 과반을 유지 → 주주총회에서 항상 이김
  → 외부 주주 압박 무력화 가능
```

이런 구조에서는 주가가 아무리 떨어져도 창업자가 원하면 그 방향으로 간다. 반대로, 이런 구조가 없으면 행동주의 투자자가 지분 5~10%만 사도 이사회를 흔들 수 있다.

## 3. 그게 실제로 성공을 일으켰나

### 성공 사례: 아마존 — 손정의 없이도 가능했던 이유

아마존의 제프 베조스는 "장기적 관점"을 끊임없이 강조했다. IPO 직후인 1997년 주주 서한에서 이미 선언했다.

> "우리는 장기적 가치 창출을 선택한다. 단기 수익성 지표는 우리의 기준이 아니다."

아마존은 20년 가까이 거의 이익을 내지 않았다. 이것이 가능했던 이유:

```
베조스의 의결권 집중 구조
→ 단기 이익을 요구하는 행동주의 투자자 압박을 막아냄
→ AWS, Prime, Alexa 등 장기 투자 지속 가능
→ 결국 AWS가 아마존 이익의 핵심이 됨

만약 외부 행동주의 투자자가 개입했다면:
→ "적자 사업 접고 배당 늘려라"
→ AWS 없는 아마존
```

베조스의 지배 구조가 아마존을 만들었다.

### 실패 사례 1: GE — 주주 환원의 끝

GE(General Electric)는 잭 웰치(Jack Welch) CEO 시절(1981~2001) "주주 가치 극대화"의 교과서였다.

```
잭 웰치 시대 GE 전략:
  매년 수익 목표 달성 압박
  목표 미달 사업부 매각 or 폐쇄
  대규모 자사주 매입
  금융 사업(GE Capital) 확대로 단기 이익 부스팅

결과 (웰치 재임 중):
  주가 40배 상승
  잭 웰치 "세기의 경영인" 칭호
```

문제는 그 다음이었다.

```
잭 웰치 퇴임 직후:
  GE Capital이 2008년 금융위기에 직격탄
  → GE 전체를 끌어내림
  → 배당 삭감, 구제금융 요청

2018년: 다우존스 지수 퇴출
2020년대: 핵심 사업부 분사, 사실상 해체

잭 웰치는 2020년 사망
→ 결과를 본인이 감당하지 않음
→ 퇴임 시 4억 달러 이상의 퇴직금 수령
```

주주 가치를 극대화한 CEO는 영웅이 됐고, 그 부작용은 다음 세대가 받았다.

### 실패 사례 2: 소프트뱅크와 위워크

손정의(孫正義)는 소프트뱅크의 창업자이자 사실상 절대 지배자다. 그의 비전은 "특이점(Singularity)", AI와 인터넷이 바꾸는 세계에 미리 투자하는 것이었다.

```
비전 펀드(Vision Fund, 2017):
  규모: 1,000억 달러 (역사상 최대 벤처펀드)
  전략: 유망 스타트업에 초대형 투자 → 시장 지배 → 수익

투자 방식:
  회사의 적정 가치보다 훨씬 높은 밸류에이션으로 투자
  → 대량 현금 주입 → 경쟁사 압살 → 독점
  → 우버, 디디추싱, 위워크에 동시 투자
```

위워크(WeWork)가 문제였다.

```
위워크 아담 뉴먼(Adam Neumann):
  "공유 오피스"를 "의식의 혁명"으로 포장
  손정의와 단 28분 미팅 → 44억 달러 투자 결정

위워크 실상:
  부동산 임대업 (기술 회사가 아님)
  수익 구조: 장기 임차 → 단기 전대 (금리 오르면 역마진)
  아담 뉴먼 사생활 유용, 자기 부동산을 회사에 임대해 수익

2019년 IPO 시도:
  밸류에이션 $470억 목표
  → 투자자들이 재무제표 보고 철수
  → IPO 철회, 뉴먼 CEO 사임

손정의 결과:
  약 $140억 손실 기록
  → 소프트뱅크 주가 급락
  → 그러나 손정의는 소프트뱅크의 최대 주주
  → 개인 재산 규모는 여전히 수조 원대
  → 퇴장 없음
```

손정의가 위워크에 밀어붙인 것은 그의 확신이었다. 틀렸지만, 그가 부담한 리스크는 그의 지분 가치 하락이었지 생존의 위협이 아니었다.

### 실패 사례 3: 사모펀드와 토이저러스

토이저러스(Toys"R"Us)는 한때 세계 최대 장난감 소매 체인이었다.

```
2005년:
  KKR, 베인 캐피털, 보나도 리얼티가 $66억에 인수
  → 차입매수(LBO, Leveraged Buyout): 빚으로 회사를 사는 방식
  → 토이저러스가 $50억의 부채를 짊어짐

이후 전략:
  아마존에 대항하는 IT 투자? → 부채 이자 내기도 버거움
  → 이자 비용 연간 수억 달러
  → 설비 투자, 인력 투자 불가
  → 아마존이 장난감 시장을 잠식하는 동안 손 놓음

2018년 파산:
  직원 33,000명 해고
  
사모펀드 수익:
  인수 과정에서 수수료 수억 달러 수령
  경영 컨설팅 명목 연간 수수료
  파산 이후 일부 부동산 매각 수익 확보
  → 토이저러스가 망해도 수익 구조는 있었음
```

회사를 산 사람이 회사를 망하게 했는데, 그 과정에서 돈을 벌었다.

## 4. 압력을 넣은 사람은 리스크를 지는가

### 구조적으로 리스크가 비대칭이다

주주, 특히 대형 기관투자자와 행동주의 투자자는 구조적으로 리스크가 제한된다.

```
행동주의 투자자의 전형적인 패턴:

1. 지분 5~10% 매입
2. 공개서한 또는 이사회 압박:
   "비핵심 사업 매각", "자사주 매입", "비용 절감"
3. 회사가 요구 수용 → 주가 단기 상승
4. 투자자 지분 매도 → 수익 실현
5. 이후 회사 장기 성과는 무관

리스크:
  틀렸을 때 손실은 투자 원금의 일부
  맞았을 때 수익은 레버리지 활용 가능
  회사가 망해도 책임은 없음
```

**개인 CEO는 다르다.** CEO가 틀리면 해고된다. 하지만 이것도 제한적이다.

```
CEO 해고의 현실:
  황금낙하산(Golden Parachute): 해고 시 수십억~수백억 원 보상
  → 틀려도 부자로 퇴장
  
  이사회 구성:
  → 주요 이사가 같은 네트워크 (대학, 전직 직장, 투자 관계)
  → "내 편"을 이사회에 심어두면 해고 어려움

예시: 위워크 아담 뉴먼
  CEO에서 쫓겨남
  → 소프트뱅크로부터 $17억 수령 (지분 매입 + 자문 계약)
  → 이후 부동산 스타트업으로 복귀, 투자 유치
```

### 책임지는 사람은 따로 있다

```
의사결정 피라미드:
  최대 주주 → 이사회 → CEO → 경영진 → 직원

리스크 방향:
  결정: 위에서 아래로
  피해: 아래에서 위로

실제 사례:
  GE 잭 웰치의 단기주의 → 2018년 구조조정 때 일반 직원 수만 명 감원
  사모펀드 LBO → 토이저러스 직원 33,000명 해고
  행동주의 투자자의 비용절감 압박 → R&D팀 해고
```

결정을 내린 사람이 보상을 받고, 결과를 감당하는 사람은 결정에 참여하지 못한 사람들이다.

### 업계에서 퇴장하는 경우는 언제인가

```
퇴장이 일어나는 경우:
  ① 본인의 돈을 직접 날릴 때
     → 창업자 자신이 대규모 지분 보유 + 레버리지
     → 엘리자베스 홈스 (테라노스) - 사기죄 유죄, 징역
     → 단, 형사 처벌까지 간 경우는 극히 드묾

  ② 규제 위반이 명확할 때
     → 회계 조작, 내부자 거래, 사기
     → 엔론, 월드컴

퇴장이 안 일어나는 경우:
  잘못된 경영 판단 (합법적 범위 내)
  → 비즈니스 판단의 원칙(Business Judgment Rule):
     법원은 경영진의 사업 판단에 개입 안 함
     설령 결과가 나빠도 고의·부정이 없으면 면책
  → 주주가 소송해도 이기기 어려움
```

합법적으로 회사를 망하게 하는 것은 범죄가 아니다.

## 5. 그렇다면 좋은 기업 지배구조는 무엇인가

이 문제에 정답은 없다. 다만 역사적으로 잘 작동한 구조들의 공통점은 있다.

### 장기 지향 지배주주 + 실력

```
워런 버핏 / 버크셔 해서웨이:
  버핏이 CEO이자 최대 주주
  → 단기 주주 압박 무력화 가능
  → "5년 뒤에 좋은 회사를 산다"는 기준 유지

제프 베조스 / 아마존:
  차등 의결권으로 지배력 유지
  → 외부 압박 없이 장기 투자 지속
  → AWS, Prime 등 장기 베팅 성공

공통점:
  지배주주가 있고
  그 지배주주가 장기 관점으로 옳은 판단을 내림
  → 이 구조가 성공한 것이지, 지배주주 구조 자체가 정답은 아님
```

### 지배주주가 잘못된 방향으로 가면

```
손정의 / 소프트뱅크:
  같은 지배주주 구조 → 위워크에 $140억 손실
  → 지배주주의 판단력이 문제

소니 짐 라이언:
  주주 압박(GaaS 수익)을 명분으로 게이머 문화 무시
  → 콩코드 실패

결론:
  지배구조가 좋다 = 지배주주가 올바른 판단을 내릴 인센티브가 있다
  → 지분을 많이 가질수록 회사 망하면 본인 손해
  → 단, 황금낙하산·수수료로 개인 이익 분리되면 인센티브 왜곡
```

---

## 정리

```
기업은 이윤을 추구한다
→ 더 정확히는: 지배주주가 원하는 방향으로 움직이고
                그 결과로 이윤이 생기거나 안 생긴다

기업을 조종하는 자:
  최대 주주 / 지배 주주 / 행동주의 투자자
  → 의결권 구조에 따라 지배력 결정
  → 차등 의결권으로 소수 지분으로도 절대 지배 가능

성공을 일으켰나:
  지배주주가 장기 관점 + 실력 있을 때: 아마존, 버크셔
  지배주주가 단기 이익 / 판단 오류일 때: GE, 위워크, 토이저러스

압력을 넣은 사람이 리스크를 지는가:
  구조적으로 아니다
  → 결정은 위에서 내리고, 피해는 아래에서 감당
  → 잘못된 경영 판단은 합법이므로 형사 책임 없음
  → 황금낙하산으로 퇴장도 부자로
  → 예외: 직접 사기·회계조작 등 형사 범죄일 때만 퇴장 강제
```

기업은 추상적인 법인이지만, 실제로는 사람이 움직인다. 그 사람이 누구냐, 그가 어떤 인센티브 구조에 놓여있냐가 회사의 미래를 결정한다. "회사가 이윤을 추구한다"는 말은 절반만 맞다. 나머지 절반은 "누구의 이윤을, 어떤 방식으로, 언제까지"다.
