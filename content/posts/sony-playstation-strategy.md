플레이스테이션은 한때 게임 산업의 방향을 정의했다. PS1의 3D 혁명, PS2의 DVD 시대, PS4의 "For The Players" 선언. 그런데 2020년대 들어 플레이스테이션은 계속 뭔가를 잘못한다. 콩코드는 역사상 최대 규모의 플로프 중 하나가 됐고, 라오어2는 팬덤을 분열시켰고, 주력 스튜디오들이 문을 닫는다. 이것은 단순한 실수의 연속이 아니다. 방향 자체가 바뀐 것이다.

---

## 1. 플레이스테이션이 지향했던 것

### PS4 시대 — "For The Players"

PS4(2013)는 명확한 정체성으로 시작했다. 마이크로소프트 Xbox One이 DRM 논란과 "TV, TV, TV" 발표로 자폭하는 사이, 소니는 E3 무대에서 단순하게 중고 게임 공유하는 방법을 보여주며(그냥 건네주면 됨) 게이머들의 환호를 받았다. 가격도 $399로 $499인 Xbox One보다 낮았다.

PS4 시대의 플레이스테이션 전략은 일관됐다:

```
퍼스트 파티 독점작 → 콘솔 구매 이유 창출
  갓 오브 워, 스파이더맨, 호라이즌, 데스 스트랜딩
  블러드본, 라오어, 언차티드 4, 고스트 오브 쓰시마

특징: 고예산, 고품질, 내러티브 중심 싱글플레이어
      영화적 연출, 공개 시 화제성
```

이것이 "PlayStation 경험"이었다. 어디서도 할 수 없는 특정 게임을 하려면 플레이스테이션을 사야 했다. 단순하고 효과적인 전략이었다.

### PS5 시대의 전략 선언 — 그리고 방향 이탈

PS5(2020) 출시 초기에는 같은 방향으로 보였다. 데모와 관련 쇼케이스에서 화제작들이 나왔다. 하지만 그 이면에서 전략이 바뀌고 있었다.

**짐 라이언(Jim Ryan)** 체제(2019~2024)에서 소니가 선언한 새 방향:

```
1. 라이브 서비스 게임 12개 (2025년까지)
   → GaaS로 지속적 수익 창출

2. PC 포팅 확대
   → 독점 해제, 더 많은 소비자에게 접근

3. 대형 인수:
   → 번지 (Bungie, Destiny 개발사): $3.6B (2022)
   → Haven Studios (GaaS 특화 신설): 설립 지원
   → Firewalk Studios (Concord 개발사) 인수

4. PS+ 개편 (구독 서비스 강화)
```

요약하면: 전통적인 "시그니처 싱글플레이어 독점작" 전략에서 **GaaS+구독+플랫폼 확장**으로 피벗이다.

문제는 이 두 전략이 공존하기 어렵다는 것이다. GaaS는 완전히 다른 개발 문화, 다른 팀 역량, 다른 사용자 관계를 요구한다.

## 2. 소니 주가가 자기 길을 가는 이유

플레이스테이션이 흔들려도 소니 그룹(Sony Group Corporation) 주가가 반드시 같이 무너지지 않는다. 이유는 소니가 플레이스테이션이 아니기 때문이다.

```
소니 그룹의 사업 부문 (2024 기준 대략):
  게임 & 네트워크 서비스 (PlayStation): 약 28%
  음악 (Sony Music): 약 13%
  영화 (Sony Pictures): 약 8%
  이미지 & 센싱 솔루션: 약 13%
    → 스마트폰 카메라 이미지 센서 (아이폰 포함)
  전자 제품 & 솔루션: 약 20%
  금융 서비스: 약 14%
  기타: 나머지
```

**이미지 센서가 핵심이다.** 소니는 전 세계 스마트폰 카메라 이미지 센서 시장에서 압도적 1위다. 애플 아이폰, 삼성, 화웨이까지 소니의 센서를 쓴다. 스마트폰 시장이 살아있으면 소니 수익의 상당 부분은 보장된다.

**Sony Music은 독립적 수익원이다.** 비욘세, 에드 시런, 아델, BTS 레이블(HYBE와 파트너십)... 음악 스트리밍 시대에 음원 권리를 가진 것은 강력한 현금 흐름이다.

```
결과:
  PlayStation에서 콩코드가 수백억 원을 날려도
  → 소니 그룹 전체 재무에서 보이는 손실은 희석됨
  → 주가는 센서 판매량, 음원 수익, 영화 흥행에 더 민감하게 반응

주주 압력의 방향:
  PlayStation에게: "GaaS로 반복 수익 만들어라"
  → 실제로 GaaS 매출이 안정적이라 주주 이익과 부합함
  → 단기적 주가 지지 → 게이머 불만은 후순위
```

이것이 플레이스테이션이 게이머보다 주주를 먼저 보는 구조다. 게이머가 아무리 불만을 가져도, 주가에 직결되지 않으면 경영진에게 압력으로 작동하지 않는다.

## 3. 무엇이 잘못됐는가 — 구조적 문제

### 잘못 #1: 번지 인수의 실패

2022년 소니는 번지(Bungie)를 36억 달러에 인수했다. 이유는 명확했다: 번지는 데스티니(Destiny)를 통해 라이브 서비스 운영 경험이 있었다. 소니에게 없는 역량이었다.

결과:

```
번지 인수 후:
  번지, 데스티니 2의 유저 이탈로 실적 악화
  번지가 소니와의 실적 목표 미달
  소니, 번지 인력의 상당수 감원 (약 220명, 2023)
  번지의 차기 프로젝트들 취소 또는 지연
  소니, 번지 매각 검토 보도까지 나옴 (2024)

36억 달러에 사서 → 수백억 원 감원 비용 + 매각 검토
```

GaaS를 하기 위해 인수한 회사가 GaaS에서 실패하고 있다. 소니는 GaaS의 DNA를 돈으로 살 수 없다는 것을 비싼 교훈으로 배웠다.

### 잘못 #2: 재팬 스튜디오 해산

2021년 소니는 재팬 스튜디오(Japan Studio)를 사실상 해산했다. 1993년 설립, 소니 플레이스테이션의 역사 그 자체였던 스튜디오다.

```
재팬 스튜디오가 만든 것들:
  ICO, 완다와 거상 (Team Ico)
  사이렌, 그라비티 러시
  아스타 봇 시리즈의 기원
  니나 개발 지원, 다수 일본 개발사 퍼블리싱 지원

해산 이유 (공식):
  "팀 아스타 봇에 집중하기 위한 재편"

실제:
  서구 시장 중심 전략에서 일본식 게임이 맞지 않았음
  짐 라이언 체제의 서구화 전략
```

팀 아스타 봇은 살아남아 아스타 봇(2024)으로 큰 성공을 거뒀다. 하지만 재팬 스튜디오 해산은 소니의 일본 게임 뿌리를 끊었다는 상징적 사건으로 남았다. 아이코(ICO) 팀은 후속작을 못 만들고 흩어졌다.

### 잘못 #3: 런던 스튜디오 폐쇄

2024년 소니는 런던 스튜디오(London Studio)도 폐쇄했다. 1997년 설립된 오래된 스튜디오다. PSVR2 게임을 주로 만들었는데, PSVR2 자체가 외면받으면서 스튜디오도 문을 닫았다.

## 4. 라오어2 사태 — 퍼스트파티의 분열

2020년 6월 출시된 **더 라스트 오브 어스 파트 2(The Last of Us Part II)**는 게임 역사상 가장 논란이 된 속편 중 하나다.

### 무슨 일이 있었나

출시 전 대규모 스포일러가 유출됐다. 내용:

```
라오어1에서 살아남은 핵심 인물 조엘이
게임 초반에 '애비'라는 캐릭터에게 사망

게임의 절반은 조엘을 죽인 애비를 플레이어로 조종하며
애비의 관점에서 이야기를 전개
```

게이머 반응은 극단적으로 갈렸다. 1편에서 조엘과 엘리의 여정에 감정적으로 투자했던 플레이어들은 이 전개에 분노했다. 반면 이것을 성숙한 내러티브로 평가하는 쪽도 있었다.

**메타크리틱 사태:**

```
평론가 점수: 93/100 (극찬)
사용자 점수 (초기): 3.4/10 (리뷰 폭탄)

역대 최대 규모의 평론가↔사용자 괴리 중 하나
```

게임이 Game of the Year를 받자 갈등은 더 깊어졌다.

### 닐 드럭만의 태도

크리에이티브 디렉터 **닐 드럭만**의 반응이 갈등에 기름을 부었다.

```
팬들의 비판 → 드럭만의 SNS 반응이 조롱으로 해석되는 사례 반복
→ "나를 싫어하는 사람들이 내 게임을 사줬다" 류의 발언
→ 비판을 진지하게 받아들이기보다 비웃는 것으로 보임
```

이것이 플레이스테이션 방관의 첫 신호였다. 소니는 퍼스트파티 크리에이터가 팬과 충돌하는 상황에서 어떤 중재도 하지 않았다. "아티스트의 자유"를 지지한다는 입장이었지만, 팬들에게는 거만함으로 읽혔다.

### 낫독의 이후 — 생산성의 멈춤

라오어2 이후 너티독(Naughty Dog)은 침묵했다.

```
라오어2 출시: 2020년 6월
현재(2026년): 후속 게임 없음

개발 중 취소된 프로젝트:
  멀티플레이어 라오어 파생작 → 수년 개발 후 취소 (2023)
  이유: "우리가 원하는 수준을 만들 수 없다"

라오어 TV 드라마(HBO): 2023년 대성공
→ 게임 업무와 TV 프로덕션이 팀 에너지를 나눔
```

소니의 가장 핵심 스튜디오가 6년째 새 게임을 못 내고 있다.

## 5. 콩코드 — 플레이스테이션 역사상 최대의 참사

**콩코드(Concord)**는 2024년 8월 23일 출시된 히어로 슈터(Hero Shooter)다. 오버워치 스타일의 팀 기반 온라인 게임이었다.

### 기본 정보

```
개발사: Firewalk Studios (소니가 2023년 인수)
개발 기간: 약 8년
추정 개발비: $200M~$400M+ (수백억~수천억 원 추정)
출시 가격: $39.99 (유료)
서비스 종료: 2024년 9월 6일

서비스 기간: 2주
```

### 왜 실패했나

**시장 분석의 완전한 실패:**

```
히어로 슈터 시장 2024년:
  오버워치 2: 무료
  발로란트: 무료
  에이펙스 레전드: 무료
  팀 포트리스 2: 무료

콩코드: $39.99

→ 같은 장르에서 무료 경쟁작이 이미 포화 상태인 시장에
  40달러를 받는 신작을 출시
```

**캐릭터 디자인 논란:**

콩코드의 캐릭터들은 출시 전부터 혹평을 받았다. 히어로 슈터는 캐릭터 아이덴티티가 핵심인 장르인데, 콩코드의 캐릭터들은 기억에 남지 않는다는 평이 지배적이었다. 오버워치의 트레이서, 겐지처럼 즉각 인식되는 실루엣이 없었다.

**출시 결과:**

```
스팀 최고 동시 접속자 수: 697명
PS5 판매량: 추정 25,000장 이하

비교:
  인디 게임도 이것보다 많이 팔림
  수백억 원을 투자한 소니 퍼스트파티 타이틀의 결과
```

출시 후 2주가 지나기 전에 소니는 판매를 중단하고 환불을 제공했다.

**그리고 Firewalk Studios 폐쇄:**

2024년 10월, 소니는 Firewalk Studios 자체를 완전히 폐쇄했다. 8년을 개발해온 회사가 게임 출시 2달도 안 돼 문을 닫았다.

### 방관의 증거

콩코드 사태에서 가장 충격적인 것은 8년 동안 아무도 멈추지 않았다는 것이다.

```
질문: 개발 중 시장 변화를 왜 반영 안 했나?
  2022년: 오버워치 2가 무료로 전환
  → 콩코드 팀은 유료 모델을 유지

질문: 베타 테스트 반응이 좋지 않았는데 왜 출시했나?
  → 출시 전 공개 베타에서 이미 접속자 수가 낮았음
  → 소니는 출시를 강행

질문: 왜 가격을 무료로 바꾸지 않았나?
  → 바꿀 수 있었지만 안 했다
```

이것은 단순한 실수가 아니다. 8년, 수백억~수천억 원의 투자를 정당화하려는 **매몰 비용의 함정**이었을 가능성이 높다. "여기까지 왔으니 출시는 해야지." 내부에서 이미 실패 신호가 보였어도 경영진이 멈추지 못했다.

## 6. 헬다이버즈 2 사태 — 성공 후의 자폭

2024년 초 헬다이버즈 2(Helldivers 2)는 소니의 기대를 훨씬 뛰어넘는 대성공을 거뒀다. 협동 슈터 장르에서 수백만 장이 팔렸다. 커뮤니티 반응도 극히 좋았다.

그런데 소니는 이 성공 직후 스스로 발을 쐈다.

```
2024년 5월:
  소니, PSN 계정 연동 의무화 발표
  → 스팀에서 구매한 플레이어도 PSN 계정 필요
  
문제:
  PSN이 서비스 안 되는 국가가 전 세계 다수
  → 그 나라 플레이어는 이제 게임을 못 함
  → 스팀 환불 요청 폭주
  → 스팀 리뷰 점수 급락 (긍정→압도적 부정)
  
소니 반응:
  → 일주일 후 정책 철회
```

헬다이버즈 2로 쌓아올린 신뢰를 한 번의 정책으로 무너뜨렸다가, 커뮤니티 반발로 철회했다. 소니의 게이머 감각이 얼마나 무뎌졌는지를 보여준 사건이었다.

## 7. 왜 방관했는가 — 구조적 해석

이 모든 사건들을 이으면 하나의 패턴이 보인다.

### 경영진과 게이머 문화의 단절

짐 라이언은 플레이스테이션의 CEO였지만 게이머 문화에 대한 감각이 없었다는 평가가 많다.

```
대표적 발언:
  Xbox-액티비전 인수 관련:
  "Deals are deals"
  → 경쟁 이슈에 대한 냉정한 비즈니스 언어

  구 게임에 대해:
  "Why would anyone play PS1 or PS2 games?"
  → 레트로게임, 하위 호환에 대한 무감각

게이머 반응: "당신이 우리를 이해 못 하는 것이 증명됐다"
```

경영진이 게이머 문화를 이해하지 못하면, 게이머가 어떤 신호를 보내도 의사결정에 반영되지 않는다.

### 지표의 함정

소니는 데이터를 본다. 그런데 어떤 데이터를 보느냐가 문제다.

```
보는 지표:
  GaaS MAU (월간 활성 사용자)
  PS+ 구독자 수
  PC 포팅 판매량
  분기 매출

안 보이는 지표:
  게이머 커뮤니티 감정
  장기적 브랜드 신뢰
  "플레이스테이션이기 때문에 사는" 충성도
```

콩코드의 베타 동시접속자가 낮아도, 경영진 보고서에 "출시 예정, 마케팅 진행 중"으로 올라가면 문제로 인식되지 않는다.

### GaaS 피벗의 문화적 부적합

소니의 강점은 내러티브 싱글플레이어였다. 이것을 잘하는 너티독, 산타 모니카, 인소나이악이 있다. 그런데 GaaS는 완전히 다른 근육이 필요하다.

```
싱글플레이어 강점:
  스토리텔링, 연출, 세계관 구축
  3~5년 개발 → 완성된 제품 출시

GaaS 필요 역량:
  운영, 커뮤니티 관리, 빠른 업데이트
  데이터 분석, 밸런싱, 이벤트 설계
  출시가 시작점, 이후가 핵심
```

소니는 돈으로 GaaS 회사(번지)를 사려 했지만, 번지 자체도 GaaS에서 흔들리고 있었다. 문화는 인수로 이식되지 않는다.

## 8. 현재와 전망

### 짐 라이언 퇴임 이후

2024년 초 짐 라이언이 퇴임하고 히로키 토토키(Hiroki Totoki)가 SIE 회장직을 맡았다. 정책 변화 신호가 일부 나타났다.

```
긍정적 변화:
  아스타 봇(2024): 재팬 스튜디오 해산 후 살아남은 팀의 게임
                    GOTY 후보급 호평, 소니 방향성 재조명
  Ghost of Tsushima PC 포팅 성공
  
여전한 문제:
  라이브 서비스 전략의 방향 불분명
  너티독의 침묵 지속
  번지 상황 불투명
```

아스타 봇의 성공은 "소니의 전통적 강점이 아직 살아있다"는 신호였다. 하지만 하나의 성공작이 전략적 혼란을 해소하지는 못한다.

### 마이크로소프트와의 비교

역설적으로 Xbox는 소니와 반대 방향에서 비슷한 고민을 하고 있다.

```
Xbox:
  게임패스로 구독 생태계 구축 → 성공
  그런데 콘솔 하드웨어 판매 부진
  → "Xbox 게임을 꼭 Xbox로 안 해도 된다"는 인식
  → 콘솔 존재 이유 약화
  → 멀티플랫폼 전략으로 전환

PlayStation:
  독점으로 콘솔 판매 → 성공
  그런데 GaaS로 수익 확장 시도 → 실패
  → PC 포팅으로 독점성 약화
  → "PS5를 꼭 사야 하나?" 질문 증가
```

두 회사 모두 자신의 전통적 강점을 스스로 희석시키는 방향으로 움직이고 있다.

## 정리

```
플레이스테이션이 지향한 것 (PS4):
  퍼스트파티 독점 → 콘솔 구매 이유 → "For The Players"

PS5 시대 실제 행보:
  GaaS 피벗 + 번지 인수 + PC 포팅 확대
  → 독점성 약화 + GaaS 실패 + 기존 강점 희석

주가가 자기 길을 가는 이유:
  Sony Group = 이미지 센서 + 음악 + 영화 + PS
  PS의 실패는 그룹 전체에 희석됨
  주주는 GaaS 반복 매출을 원함 → 방향 지지

구조적 잘못:
  ① 번지 인수: GaaS DNA는 돈으로 살 수 없었음
  ② 재팬스튜디오 해산: 정체성의 뿌리를 제거
  ③ 콩코드: 시장 분석 실패 + 8년 매몰 비용 함정
  ④ 라오어2 이후: 너티독 침묵 6년째
  ⑤ 헬다이버즈 PSN 사태: 성공 직후 자폭

방관의 원인:
  경영진의 게이머 문화 단절
  보이는 지표(MAU, 구독자)와 안 보이는 신뢰 사이의 괴리
  GaaS 피벗이 만든 문화적 부적합
  주가와 게이머 감정의 비연동
```

플레이스테이션의 위기는 하나의 나쁜 결정이 아니다. 강점을 알고 그것을 지키는 것보다, 더 큰 수익 구조를 찾으려다 강점 자체를 잃어버리는 과정이다. 게이머들이 "뭔가 달라졌다"고 느끼는 것은 옳다. 실제로 달라졌다.
