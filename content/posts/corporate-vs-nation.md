사이버펑크 세계관에서 기업은 국가를 대체한다. 군대를 보유하고, 법을 만들고, 시민을 통제한다. 블레이드 러너의 타이렐 코퍼레이션, 에이리언의 웨이랜드-유타니, 사이버펑크 2077의 아라사카. 모두 픽션이지만 완전한 상상은 아니다. 역사에는 실제로 군대를 보유하고 영토를 지배한 기업이 있었다. 그리고 지금도 어떤 기업들은 특정 국가보다 더 큰 경제적 힘을 갖고 있다.

기업이 국가보다 강할 수 있는가? 정답은 "상황에 따라 이미 그렇다"이다.

---

## 1. 국가의 힘이란 무엇인가

비교를 하려면 먼저 "국가의 힘"이 무엇인지를 정의해야 한다.

사회학자 막스 베버(Max Weber)는 국가를 이렇게 정의했다.

> "특정 영토 내에서 **물리적 강제력의 합법적 독점**을 갖는 인간 공동체"

핵심은 두 가지다.

```
국가 권력의 두 기둥:
  ① 폭력의 합법적 독점
     → 군대, 경찰, 사법권
     → 다른 누구도 합법적으로 총을 쏠 수 없음

  ② 영토와 주권
     → 특정 공간에서의 지배권
     → 국경 안에서 법이 적용됨
```

이 정의에 따르면 기업은 국가가 될 수 없다. 폭력을 합법적으로 독점하지 못하고, 영토를 주권적으로 지배하지 못한다.

그런데 역사는 예외를 만들어왔다.

## 2. 실제로 국가를 능가했던 기업들

### 동인도회사 — 기업이 제국이 된 유일한 사례

**영국 동인도회사(East India Company, 1600～1858)**는 인류 역사에서 기업이 국가에 가장 근접했던 사례다.

```
설립: 1600년, 영국 왕실의 무역 독점권 부여
초기 목적: 인도·동남아시아와의 향신료 무역

이후 전개:
  1757년 플라시 전투 승리
  → 벵골(현 방글라데시+인도 동부) 통치권 획득
  → 세금 징수권, 사법권, 군사권 행사

최전성기 동인도회사:
  보유 병력: 약 260,000명 (영국 정규군의 2배)
  통치 영토: 인도 아대륙 대부분
  세금 징수: 직접 징수
  법원 운영: 직접 재판
  화폐 발행: 일부 지역에서 독자 통화
  외교: 독립적으로 조약 체결
```

동인도회사는 기업이면서 사실상 제국이었다. 주주들은 배당을 받았고, 인도 농민들은 세금을 냈다. 무역 회사가 군사 정복을 통해 국가 기능을 흡수한 것이다.

1857년 세포이 항쟁(인도 대반란) 이후 영국 왕실이 직접 통치를 선언하며 회사는 해산됐다.

### 유나이티드 프루트 컴퍼니 — 바나나 공화국을 만든 기업

**유나이티드 프루트 컴퍼니(United Fruit Company, 현 Chiquita)**는 중앙아메리카를 사실상 지배했다.

```
1900년대 초 과테말라, 온두라스, 코스타리카, 콜롬비아:
  UFC 소유 토지: 각국 최대 농경지
  UFC 소유 철도: 국가 물류의 핵심
  UFC 소유 항구: 수출입 창구 독점
  UFC 소유 전신망: 국가 통신 인프라

결과:
  이 나라들의 정부는 UFC의 요구를 거절 못함
  → 세금 면제 요구 수용
  → 토지 수용 반대 불가
  → 노동자 보호법 제정 차단

"바나나 공화국(Banana Republic)":
  UFC의 이익을 위해 정치가 결정되는 나라
  → 영어 단어로 정착 (허약한 정부 + 단일 외국기업 지배)
```

1954년 과테말라의 아르벤스 대통령이 UFC 토지를 국유화하려 하자, UFC의 로비를 받은 미국 CIA가 쿠데타를 지원해 정권을 교체했다. 기업의 이익을 위해 국가 정부가 교체된 것이다.

### 벨기에령 콩고 — 개인 소유 국가

엄밀히는 기업이 아니지만, 레오폴드 2세의 콩고 자유국(Congo Free State, 1885～1908)은 한 개인이 영토를 사유재산처럼 지배한 사례다.

```
레오폴드 2세:
  벨기에 왕이면서 개인 자격으로 콩고를 "소유"
  → 국가가 아닌 개인 기업처럼 운영
  → 고무 생산 할당량 강제
  → 미달 시 손목 절단, 학살
  → 추정 사망자 1,000만~1,500만 명

구조:
  레오폴드 개인 → 콩고 지배 → 이익 수취
  국제 사회가 학살 드러낼 때까지 20년간 지속
```

이것은 기업이 국가를 능가한 사례가 아니라, 국가 권력 자체가 사유화된 사례다. 기업과 국가의 경계가 완전히 무너진 극단이다.

## 3. 현재: 기업이 국가보다 강한 영역들

지금 시대에 기업이 군대를 가지고 영토를 지배하는 경우는 없다. 하지만 특정 영역에서는 국가보다 강한 힘을 행사한다.

### 경제 규모 — GDP와 시가총액 비교

```
2024년 기준 기업 시가총액 vs 국가 GDP 비교:

애플:         $3.5조 (3,500,000,000,000달러)
사우디아라비아 GDP: $1.1조

마이크로소프트: $3.2조
독일 GDP:      $4.5조 (비슷한 규모)

엔비디아:     $3.0조
프랑스 GDP:   $3.0조

구글(알파벳): $2.2조
이탈리아 GDP: $2.1조
```

시가총액이 GDP와 직접 비교되진 않는다. GDP는 연간 생산액이고 시가총액은 미래 이익의 할인 현재가치다. 하지만 상징적으로, 한 회사가 한 나라의 연간 경제 산출량에 필적하는 가치를 갖는다는 것은 그 자체로 의미가 있다.

### 데이터와 정보 권력

```
메타(Facebook)의 사용자: 약 30억 명
지구 인구: 80억 명
→ 지구 인구의 37%가 메타에 개인정보를 제공

구글의 검색 점유율: 약 90%
→ 전 세계 대부분의 사람이 "무엇을 검색해야 하는지"를
  구글 알고리즘이 결정

결과:
  어떤 국가도 30억 명의 행동 패턴·관심사·관계망을
  메타만큼 자세히 알지 못함
  
  국가는 자국민 데이터를 메타에 요청해야 함
  → 반대로 메타가 데이터를 정치에 활용한 사례:
     케임브리지 애널리티카 스캔들 (2016 미국 대선 영향)
```

정보는 현대의 권력이다. 데이터를 가진 기업은 그것을 가지지 못한 국가보다 더 많은 것을 안다.

### 조세 회피 — 국가 주권을 무력화하는 합법적 수단

```
구글의 "더블 아이리시 위드 더치 샌드위치":
  아일랜드 법인 A → 네덜란드 법인 → 아일랜드 법인 B (버뮤다 거주)
  → 이익이 세율 0%인 버뮤다에 축적
  → 수십억 달러 합법적 세금 회피

애플:
  아일랜드 법인세율 12.5%를 이용
  유럽 이익 대부분을 아일랜드에 귀속
  유럽 각국 정부: 세금 못 걷음

결과:
  EU가 애플에게 $160억 추가 세금 부과 명령 (2016)
  → 애플 소송 → 수년간 법정 다툼
  → 2024년 유럽 최고법원에서 EU 승소, 애플 납부 명령

교훈:
  국가가 세금을 걷으려면 수년의 소송이 필요
  그 기간 동안 기업은 이익을 계속 쌓음
  → 국가의 세금 주권이 기업 법무팀에 의해 지연됨
```

### 민간군사기업(PMC) — 합법적 폭력의 외주화

```
블랙워터 (현 Academi):
  이라크 전쟁 당시 미국 국무부 경호 담당
  최전성기 직원: 약 90,000명 (전 세계)
  2007년 니수르 광장 민간인 17명 사살 사건

바그너 그룹 (러시아):
  리비아, 시리아, 말리, 중앙아프리카, 우크라이나 활동
  러시아 정부와 공식·비공식 관계
  국가가 공식적으로 시인하기 어려운 작전 수행

기업이 폭력을 행사하는 방식:
  국가의 위탁 → 민간군사기업 → 전장
  국가는 책임 회피 가능
  기업은 법적 공백 지대에서 활동
```

동인도회사처럼 군대를 직접 보유한 것은 아니지만, 국가에서 폭력을 외주받아 행사하는 구조는 존재한다.

### 빅파마 — 생명권에 대한 지배

```
인슐린 가격 사례 (미국):
  1920년대 발견 당시 특허 $1에 토론토 대학에 양도
  → "모든 사람이 접근할 수 있어야 한다"는 원칙

  2023년 미국 인슐린 가격:
  릴리, 노보 노르디스크, 사노피 3사 독과점
  → 동일 제품이 캐나다의 10~20배 가격

  당뇨 환자 인슐린 배급:
  "한 달치만 살 수 있어서 한 번에 다 못 씀"
  → 인슐린 부족으로 사망하는 미국인 존재

국가(미국 정부)의 대응:
  수십 년간 가격 규제 불가 (로비에 의해 입법 차단)
  2022년 인플레이션 감축법(IRA)으로 일부 규제 시작
  → 빅파마 로비에 의해 축소된 형태로 통과

결론:
  일부 의약품에서는 기업이 시민의 생사를
  국가보다 더 직접적으로 좌우함
```

## 4. 국가가 기업에 굴복하는 메커니즘

기업이 국가를 직접 제압하는 것이 아니다. 국가가 스스로 굴복하는 구조가 있다.

### 고용과 세금 — 경제적 인질

```
삼성과 한국:
  삼성 그룹 매출: 한국 GDP의 약 20%
  삼성 계열사 시가총액: 코스피 시장의 30% 이상
  삼성 수출 비중: 한국 총수출의 약 20%

결과:
  한국 정부가 삼성을 강하게 규제하면?
  → 삼성이 해외 이전하면 국가 경제에 치명타
  → 정치적 부담

이재용 재판 사례:
  횡령·뇌물 유죄 판결 → 실형
  → 경제에 미치는 영향 고려해 가석방·사면 반복
  → "삼성 리스크"라는 개념 자체가 기업의 사법 면역을 시사
```

### 로비와 회전문 인사

```
미국 국방부와 방산 기업:
  록히드 마틴, 레이시온, 보잉
  → 전직 국방부 관료가 방산기업 임원으로
  → 전직 방산기업 임원이 국방부 관료로
  → "회전문(Revolving Door)"

결과:
  국방 예산 결정 → 전 직장 또는 전 부하가 수혜자
  방산 기업 로비 예산: 연간 수억 달러
  → 예산 삭감이 정치적으로 불가능한 구조

실리콘밸리와 규제:
  구글, 메타, 아마존 로비 지출: 연간 수억 달러
  FTC(연방거래위원회) 독점 소송 → 10년 이상 지연
  → 소송 중에도 기업은 성장
```

### 국제 중재 — 국가 법을 우회하는 기업의 무기

```
양자투자협정(BIT)과 ISDS:
  ISDS(투자자-국가 분쟁 해결): 
  기업이 국가의 정책에 손해를 입으면
  → 국제 중재 기관에 소송 가능
  → 해당 국가 법원을 우회

실제 사례:
  필립모리스 vs 우루과이 (담배 경고 그림 강제):
  → 우루과이 정부 소송
  → 우루과이 GDP의 수% 규모 배상 요구
  → 우루과이가 이겼지만 소송 비용만 수천만 달러

  엑손모빌 vs 베네수엘라 (석유 국유화):
  → ICSID 중재 → 베네수엘라에 배상 명령

결과:
  개발도상국이 외국 기업에 불리한 정책을 펴면
  → 천문학적 배상 위험
  → 사실상 정책 선택권 제한
  → "규제 냉각 효과(Regulatory Chill)"
```

## 5. 사이버펑크 시나리오는 실현 가능한가

창작물의 메가코프는 국가를 완전히 대체한다. 이것이 현실이 되려면 무엇이 필요한가.

```
사이버펑크 메가코프의 조건:
  ① 국가 군사력 능가
  ② 독자적 영토·법 체계
  ③ 시민이 국가 대신 기업에 종속

현재 상황과의 거리:
  ① 군사력: 핵무기를 가진 국가를 능가할 기업은 없음
              단, PMC는 일부 약소국 군사력 수준에 근접
  ② 영토·법: 없음. 단, 조세 피난처·국제 중재로 주권 우회는 가능
  ③ 종속: 일부 플랫폼 노동자는 사실상 기업 규칙이 국가 법보다 더 직접적으로 삶을 결정
```

완전한 메가코프는 현실에서 멀다. 하지만 조건이 갖춰지는 방향이 있다.

### 기업이 더 강해지는 조건

```
국가가 약해질 때:
  재정 파산 (아르헨티나, 짐바브웨, 스리랑카)
  → 외국 기업에 자원 양허, 인프라 매각
  → 중국 기업의 스리랑카 항구 99년 임차

내전·실패 국가:
  소말리아: 정부보다 씨족+외국 기업이 더 강한 통제력
  콩고: 광산 지역에서 기업+무장세력이 사실상 통치

기술 독점:
  인터넷 인프라, AI, 위성통신을 단일 기업이 독점하면
  → 스타링크가 우크라이나 전쟁에서 전략 자산화
  → 일론 머스크가 스타링크 접속을 정치적으로 활용한 정황 보도
```

### 지금도 기업이 국가보다 강한 현실적 사례

```
아마존 본사 입찰 (HQ2, 2018):
  아마존이 제2 본사 부지 모집
  → 238개 도시가 세금 감면, 인프라 지원 패키지 경쟁
  → 뉴욕시: $30억 세금 혜택 제안
  → 아마존이 "국가들 사이에서" 가장 유리한 조건 선택

결과:
  기업이 조건을 제시하고 국가(도시)가 경쟁하는 구조
  → 사실상 기업이 갑, 국가가 을
```

## 6. 왜 완전한 메가코프는 아직 없는가

반론도 있어야 한다. 기업이 국가를 완전히 대체하지 못하는 이유.

```
① 폭력의 한계:
   핵을 가진 국가를 군사적으로 이길 수 없음
   민간군사기업은 결국 국가의 허가 하에 작동

② 정당성(Legitimacy)의 부재:
   국가는 "국민의 동의"라는 명분을 갖고 있음
   기업은 계약 관계 → 불만족시 이탈 가능
   메가코프가 되려면 이탈 자체를 막아야 함 → 강제력 필요 → 국가가 됨

③ 기업 내부의 주주 압력:
   주주는 단기 이익을 원함
   영토 지배와 군사 유지는 엄청난 비용
   "군대 유지하면 분기 실적이 나빠짐"
   → 기업이 스스로 국가화하려는 인센티브가 약함

④ 국가 간 경쟁으로 기업 견제:
   미국이 구글을 규제 못 해도, EU가 규제
   중국이 자국 내 메가코프를 키워도 외국 기업은 차단
   → 단일 글로벌 메가코프는 국가 간 이해충돌로 막힘
```

## 정리

```
역사적으로 기업이 국가를 능가한 사례:
  영국 동인도회사: 군대 26만명, 인도 영토 지배
  유나이티드 프루트: 중앙아메리카 바나나 공화국 조종
  → 이것은 가능했고, 실제로 일어났다

현재 기업이 국가보다 강한 영역:
  경제 규모: 일부 기업 시가총액 > 중소국 GDP
  데이터: 어떤 국가도 메타·구글만큼 시민을 모르는 나라 있음
  조세: 기업이 국가의 세금 주권을 합법적으로 우회
  로비: 국가 정책을 기업이 설계하는 구조
  ISDS: 국제 중재로 국가 입법권 사실상 제한

기업이 국가를 완전히 대체하지 못하는 이유:
  핵무기 앞에서 군사력 열세
  정당성(국민 동의) 부재
  단기 이익 추구하는 주주 구조 → 국가 유지 비용 감당 불가
  국가 간 경쟁으로 글로벌 독점 차단

현실적 결론:
  사이버펑크 메가코프는 아직 없다
  하지만 특정 조건(실패 국가, 기술 독점, 자원 의존)에서
  기업은 이미 국가보다 강한 영향력을 행사한다
  동인도회사는 픽션이 아니라 역사였다
  그리고 그것은 단 하나의 무역 회사로 시작했다
```

창작물의 메가코프가 순수한 상상이 아닌 이유가 여기에 있다. 과거에 이미 일어났고, 현재에도 부분적으로 일어나고 있다. 완전한 형태로는 아직 없지만, 방향은 그쪽으로 기울어 있다. 차이는 단지 국가가 얼마나 버티느냐다.
