"경제가 3% 성장했다"는 뉴스를 들을 때, 이것이 개인의 최대치와 관계가 있는 말인지 의아하게 느껴질 수 있다. GDP 성장률은 평균이다. 그런데 평균이 개인의 상한선이 되려면, 소득이 모든 사람에게 균등하게 분배돼야 한다. 현실은 그렇지 않다.

경제 성장률은 개인의 평균 최대치가 아니다. 어떤 사람에게는 상한이고, 어떤 사람에게는 출발선이며, 어떤 사람에게는 관계없는 숫자다.

---

## 1. GDP 성장률이 "평균"이라는 것의 의미

### GDP와 개인 소득의 관계

```
GDP (국내총생산):
  한 나라에서 1년간 생산된 재화와 서비스의 총액
  성장률 3% = 지난해보다 3% 더 생산됨

1인당 GDP:
  GDP ÷ 인구
  → 나라가 3% 성장했다면 평균적으로 1인당 소득도 3% 증가?
  → 수학적으로는 그렇지만, 실제 분배는 다르다
```

1인당 GDP는 평균이다. 그리고 평균은 분포가 고르지 않으면 아무것도 말해주지 않는다.

```
예시:
  10명이 있는 나라의 소득:
  9명: 연 3,000만원
  1명: 연 10억원

  평균 소득: (9 × 3,000 + 10,000) / 10 = 3,700만원

  "평균이 3,700만원이다" → 9명의 현실을 반영 못함
  "GDP가 3% 성장했다" → 누가 그 성장분을 가져갔는지 말 안 함
```

실제로 지난 40년간 많은 나라에서 GDP는 성장했지만 중위 소득(상위 50%의 경계)은 거의 오르지 않았다. 성장분의 상당 부분이 상위 계층에 집중됐기 때문이다.

### 왜 성장이 균등하지 않은가 — r > g

경제학자 토마 피케티(Thomas Piketty)는 2013년 『21세기 자본』에서 이 불균형의 구조를 수식 하나로 설명했다.

```
r > g

r: 자본 수익률 (자산을 보유했을 때 얻는 수익)
g: 경제 성장률 (GDP 증가율)

역사적 수치:
  r ≈ 4~5% (배당, 임대료, 이자, 자본 이득)
  g ≈ 1~3% (GDP 성장률)

의미:
  자본을 가진 사람의 부는 경제 성장보다 빠르게 늘어남
  → 자본 없이 노동만 하는 사람은 경제 성장률만큼 올라가거나
    그것에도 못 미침
  → 자본을 가진 사람은 경제 성장률을 초과해 부가 축적
  → 시간이 지날수록 격차 확대
```

경제 성장률(g)은 **노동 소득자의 대략적인 상한**에 가깝다. 자본 소득자는 그 위에 있다.

## 2. 소득의 종류가 상한을 결정한다

개인이 얼마나 높이 올라갈 수 있는지는 **어떤 종류의 소득**을 갖느냐에 따라 구조적으로 다르다.

### 노동 소득 — 시간이 상한

```
노동 소득 = 시간 × 시간당 가치

한계:
  하루 24시간 고정
  → 아무리 잘해도 일할 수 있는 시간의 한계가 있음

의사 A: 시간당 30만원, 하루 10시간 진료
  연 수입 = 30만 × 10h × 250일 = 7억 5천만원
  → 이 이상은 물리적으로 불가능

뛰어난 의사도, 뛰어난 변호사도, 뛰어난 컨설턴트도
  "레버리지" 없으면 시간의 벽을 넘지 못함

레버리지:
  직원 채용: 다른 사람의 시간을 쓴다
  → 의사가 병원 법인을 만들면 자신이 진료 안 해도 수익 발생
  → 이 순간부터 노동 소득이 자본 소득으로 전환
```

### 자본 소득 — 이론적 상한 없음

```
자본 소득 = 보유 자산 × 수익률

특성:
  내가 자는 동안에도 돌아감
  규모가 커질수록 더 빠르게 커짐 (복리)
  시간의 벽 없음

복리의 위력:
  1억원 × 연 7% 복리:
  10년 후: 약 1.97억
  20년 후: 약 3.87억
  30년 후: 약 7.61억
  40년 후: 약 14.97억

  원금은 1억, 40년 후 약 15배
  → 일한 것이 아니라 "가지고 있던 것"의 결과
```

자본 소득의 수익률(r)이 경제 성장률(g)보다 크면, 자산이 많을수록 경제 성장에서 점점 더 많은 몫을 가져가는 구조다.

### 네트워크 소득 — 한계비용 제로의 세계

디지털 시대에는 세 번째 카테고리가 생겼다.

```
한계비용 제로(Zero Marginal Cost) 제품:
  소프트웨어, 플랫폼, 콘텐츠

Windows를 한 명 더 판매하는 비용: 0원 (추가 CD 없음)
  → 100만 명에게 팔아도 추가 제조비 없음

유튜브 영상을 한 명 더 보는 비용: 거의 0
  → 조회수 1 vs 1억의 창작 비용 차이 없음

결과:
  사용자가 늘어도 비용은 안 늘고 수익은 늘어남
  → 이익률이 규모에 비례해 폭발적으로 상승

네트워크 효과:
  사용자가 많아질수록 플랫폼 가치 상승
  → 카카오톡: 사용자가 많아서 더 쓸 수밖에 없음
  → 강자는 더 강해지는 구조 (승자독식)
```

이것이 왜 소프트웨어 기업가가 유례없는 부를 축적하는지의 이유다. 빌 게이츠, 저커버그, 베조스의 공통점은 한계비용이 거의 없는 사업을 만든 것이다.

## 3. 소득 분포는 종형 곡선이 아니다

학교에서 배우는 통계적 사고는 "평균 근처에 대부분이 몰린다"는 정규분포다. 하지만 소득과 부의 분포는 다르다.

### 멱함수 분포 (Power Law)

```
신장 분포 (정규분포):
  평균 170cm 근처에 대부분
  220cm인 사람은 극히 드묾
  최대 신장 ≈ 평균의 1.5배 수준

소득 분포 (멱함수):
  중위 소득: 약 3,500만원 (한국 기준)
  상위 0.1%: 수십억원
  최상위: 수십조원
  최대 소득 ≈ 평균의 수천~수만 배

이것이 멱함수 분포:
  소수가 극단적으로 많이 가지고
  대다수는 평균 이하
  → "평균이 상한"이라는 직관이 틀림
```

### 파레토 법칙

```
빌프레도 파레토(1896)의 관찰:
  이탈리아 토지의 80%를 인구의 20%가 소유

현대 부의 분포 (2024년 기준):
  전 세계 상위 1%: 전체 부의 약 45% 보유
  전 세계 상위 10%: 전체 부의 약 76% 보유
  하위 50%: 전체 부의 약 2% 보유

한국:
  상위 10%의 순자산: 하위 50%의 약 40배 수준
```

분포가 이렇게 비대칭이면, "경제 성장률 = 개인 상한"은 틀린 전제다. 경제가 3% 성장할 때 상위 1%의 자산은 그보다 훨씬 빠르게 늘고, 하위 계층은 실질적으로 줄어들 수도 있다.

## 4. 역사상 개인이 찍은 최대치

### 절대적 기준: 록펠러

```
존 D. 록펠러 (John D. Rockefeller, 1839~1937):
  스탠더드 오일로 미국 석유 시장의 90% 장악
  최전성기 자산: 당시 미국 GDP의 약 1.5%

현재 환산:
  2024년 미국 GDP 기준으로 환산하면 약 4,000억 달러 이상
  → 현재 머스크·베조스보다 더 큰 규모로 추정

가능했던 이유:
  물리적 독점 (석유 유통망 전체 장악)
  당시 반독점법 없음
  미국 산업화 초기 - 비어있는 시장
```

### 현대: 디지털 독점

```
일론 머스크 (2024 기준 약 $300B+):
  테슬라: 전기차 시장 선점
  스페이스X: 민간 우주 사실상 독점
  스타링크: 위성 인터넷 지배
  X(트위터), xAI 등

제프 베조스 (약 $200B):
  아마존: e커머스 + AWS (클라우드)
  한계비용 제로의 AWS가 이익의 핵심

마크 저커버그 (약 $180B):
  메타: 30억 명의 네트워크 독점
  광고 플랫폼의 네트워크 효과

공통점:
  플랫폼 또는 네트워크 효과 기반
  한계비용이 매우 낮거나 제로
  글로벌 시장 (국내 시장 한계 없음)
  독점적 위치 (경쟁자가 따라잡기 어려운 구조)
```

### 한국 기준

```
이건희 (삼성):
  별세 직전 자산 약 26조원 (2020)
  삼성 지배구조의 정점

정주영 (현대):
  빈손에서 시작, 현대그룹 창업
  → 레버리지(차입) + 국가 성장과 함께
  → 한국 경제 성장률이 15~20%일 때 그 흐름을 탄

이재용:
  상속 + 경영 승계

한국 최상위의 특징:
  대부분 재벌 창업 1세대 or 상속
  한국 경제 고성장기(1970~1990)에 기반 구축
  → "경제 성장보다 빠르게" 간 것이 아니라
    "경제 성장 자체를 만든 사람들"
```

## 5. 평범한 개인은 어디까지 찍을 수 있나

"재벌이나 빌 게이츠 얘기는 나랑 관계없다"면, 현실적으로 개인이 경제 성장률을 어떻게 초과할 수 있는지를 봐야 한다.

### 자산 시장 — 경제 성장률 초과는 가능하다

```
미국 S&P 500 지수 장기 수익률 (1928~2024):
  연평균 약 10% (배당 포함)
  인플레이션 제외 실질 수익률: 약 7%

미국 GDP 성장률 (같은 기간):
  연평균 약 3.2%

즉: 미국 주식 시장 = GDP 성장률의 약 2~3배

한국 코스피 장기 수익률:
  변동성 크고 기간별 편차 있음
  외환위기, 금융위기 등 이벤트 포함하면 복잡
  → 글로벌 분산 투자가 일반적으로 유리
```

**왜 주식이 GDP보다 빠른가?**

```
기업 이익 성장 > GDP 성장:
  GDP는 경제 전체 (비효율 포함)
  주식은 이익을 내는 기업만 선별
  → 이익 성장률이 경제 성장률보다 높을 수 있음

국제 분산:
  한국 주식만 → 한국 경제에 묶임
  글로벌 ETF → 전 세계 최고 기업들의 성장에 참여
  → 지구 어딘가가 성장하면 이익
```

### 복리 + 시간 = 개인이 찍을 수 있는 현실적 최대치

```
월 100만원 투자, 연 7% 수익 가정:

10년: 약 1.74억
20년: 약 5.24억
30년: 약 12.2억
40년: 약 26.5억

투입 원금:
  40년 × 12개월 × 100만원 = 4.8억

자산: 26.5억 (원금의 5.5배)
이자·수익: 21.7억

"경제 성장률 3%로 계산하면?"
  40년, 동일 조건: 약 9.2억
  → 7% 수익 시의 절반도 안 됨
```

복리에서 수익률 차이는 시간이 갈수록 기하급수적으로 벌어진다. 경제 성장률(3%)과 주식 시장 장기 수익률(7%)의 차이가 40년 후 3배 가까운 자산 차이를 만든다.

### 사업 소득 — 레버리지가 상한을 없앤다

```
노동 소득의 한계: 시간 × 시간당 가치
사업 소득: 직원 수 × 시간 × 시간당 가치 (내 시간 없어도)

더 중요한 것:
  내가 만든 시스템이 돌아가는 구조
  → 소프트웨어, 프랜차이즈, 콘텐츠, 플랫폼

사례:
  앱 개발자:
  → 앱 하나 만드는 데 3개월
  → 출시 후 매달 수익 발생 (내가 자는 동안)
  → 이것이 "자본 소득화된 노동"

  유튜버:
  → 영상 하나 제작: 10시간
  → 이후 5년간 조회수 발생
  → 제작 시간 대비 수익이 기하급수적

한계:
  대부분의 앱, 대부분의 유튜브 채널은 수익이 없음
  → 상위 1~5%만 실질적 수익
  → 성공 가능성과 기대 수익의 분포 = 또 다른 멱함수
```

## 6. 구조가 개인 상한을 결정한다

개인이 찍을 수 있는 최대치는 타고난 능력보다 **어떤 구조에 놓여있는가**에 더 많이 달려있다.

```
같은 능력, 다른 결과:

능력 동일한 두 사람:
  A: 대기업 직원 → 연봉 1억 상한이 현실적 최대
  B: 같은 능력으로 SaaS 제품 창업 → 상한 없음 (0이 될 수도)

같은 시장, 다른 타이밍:
  1990년대 한국 부동산 → 구매만 해도 수십 배
  2020년대 한국 부동산 → 이미 올라있고 레버리지 규제

구조가 만드는 차이:
  어느 나라에서 태어났는가: 성장률, 시장 접근성 다름
  어느 시대에 활동했는가: 한국 고성장기 vs 저성장기
  어떤 자산 클래스에 접근했는가: 부동산, 주식, 기술 창업
  어떤 네트워크를 갖는가: 투자, 인재, 정보 접근
```

### 시대가 상한을 결정하는 이유

```
조선 시대 조선인:
  아무리 능력 있어도 세계 시장에 접근 불가
  → 조선 경제 성장률이 개인의 진짜 상한

현대 개인:
  인터넷으로 전 세계 시장에 접근
  → 한국 GDP 성장률이 상한이 아님
  → 전 세계 성장 + 자본 시장에 참여 가능

디지털 네이티브 창업자:
  서울에서 만든 앱이 미국에서 팔림
  → 한국 경제 성장률은 무관
  → 글로벌 TAM(전체 시장 규모)이 상한의 척도
```

## 7. 경제 성장률이 실제로 상한이 되는 경우

경제 성장률이 개인의 실질적 상한이 되는 상황도 있다.

```
① 노동만으로 소득을 얻을 때
  → 임금 상승률 ≈ 경제 성장률 (장기적으로)
  → 자본 없이 노동만: 경제 성장률이 대략적 상한

② 정체된 시장에서 일할 때
  → 파이가 안 커지는 시장: 내가 더 가져가면 남이 덜 받음
  → 영합 게임: 평균이 최대치에 가까움

③ 규제·독점이 진입을 막을 때
  → 개인의 상향 이동이 구조적으로 차단됨
  → 특권 계층이 자본 소득을 독점

④ 인플레이션이 수익을 먹을 때
  → 예금 이자율 < 인플레이션
  → 돈을 "안전하게" 갖고 있는 것이 실질적으로 손해
  → 경제 성장률이 상한이 아니라 "은행 예금이 손해"
```

---

## 정리

```
경제 성장률이 개인의 평균 최대치인가?
  → 아니다. 다음 세 경우를 구분해야 한다:

  1. 노동 소득만 있는 경우:
     경제 성장률 ≈ 임금 상승률의 장기 상한
     → 맞다. 경제 성장률이 대략적 상한

  2. 자본 소득이 있는 경우:
     자본 수익률(r) > 경제 성장률(g) (피케티)
     → 경제 성장률보다 높은 수익 구조적으로 가능
     → 글로벌 주식 장기 수익률 ~7% vs 경제 성장률 2~3%

  3. 네트워크/플랫폼 소득:
     한계비용 제로 + 네트워크 효과
     → 이론적 상한 없음 (머스크, 베조스의 영역)

개인이 찍을 수 있는 최대치:
  현실적: 글로벌 지수 투자 × 복리 × 시간
           → 경제 성장률의 2~3배 수익 장기적으로 가능
  사업적: 레버리지 × 한계비용 낮은 모델 × 시장 규모
           → 상한 없음, 하한도 없음 (0 포함)
  구조적: 시대, 나라, 접근 가능한 자산 클래스
           → 개인 능력보다 구조가 더 크게 결정

가장 중요한 것:
  "경제 성장률 3%" 뉴스는 평균이다
  평균 아래에 있는 사람에게 상한선이고
  자본을 움직이는 사람에게는 출발선이다
  그리고 그 차이는 "어떤 소득을 갖느냐"에서 시작한다
```

경제 성장이 모두에게 균등하게 흘러갔다면, 성장률이 개인 상한이 맞다. 하지만 r > g가 작동하는 세계에서, 성장의 과실은 자본 쪽으로 더 많이 흘러간다. 경제 성장률을 개인의 상한으로 볼 것인가, 최솟값으로 볼 것인가는 어떤 소득 구조에 있느냐가 결정한다.
